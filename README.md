# ups-client

The GUI is implemented in Python.

---

## Requirements

- macOS / Linux / Windows
- Python **3.12 or 3.13**
  - ⚠️ `pygame` is currently **not compatible with Python 3.14**
- `pip`
- `venv`

---

## Recommended Setup (pyenv)

If you use multiple Python versions, `pyenv` is recommended.

### Set Python version for this project

```bash
pyenv local 3.12.8
